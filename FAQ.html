FAQ

What is SEND?

SEND is a secure encryptor and decryptor that allows you to encrypt and decrypt messages using a secret key.

How to use SEND?

To use SEND, you need to have a secret key. You can generate a secret key using the generate key button.

How to generate a secret key?

To generate a secret key, you need to click the generate key button.

How to encrypt a message?

To encrypt a message, you need to enter the message you want to encrypt and the secret key. Then click the encrypt button.

How to decrypt a message?

To decrypt a message, you need to enter the message you want to decrypt and the secret key. Then click the decrypt button.

How to copy the encrypted/decrypted message?

To copy the encrypted/decrypted message, you need to click the copy button.

What are files that can be encrypted/decrypted?

Files that can be encrypted/decrypted are Text files, Microsoft Word, Excel, and PowerPoint files, images, videos, and audio files.

What is the maximum file size that can be encrypted/decrypted?

The maximum file size that can be encrypted/decrypted is 10 MB for audio and video files, text files, Microsoft Word, Excel, and PowerPoint files, and 15 MB for images files.
